# Landing Starter Pack (DE/EN + Dark/Light)

Dieses Paket ist ein neutrales Grundgeruest fuer neue Landingpage-Varianten.

## Dateien
- `locales/de.json`
- `locales/en.json`
- `theme.css`
- `i18n.js`
- `theme.js`
- `template.html`

## Nutzung
1. `template.html` als Basis in ein neues HTML kopieren.
2. `theme.css`, `i18n.js`, `theme.js` einbinden.
3. Texte ueber `data-i18n` aus den Locale-Dateien laden.
4. Farben nur ueber CSS-Variablen steuern.
